# Claude Code 快速入门

Claude Code 是 Anthropic 官方出品的 AI 编程助手，以命令行工具的形式运行，可以读写代码、执行命令、帮你完成复杂的编程任务。

---

## 安装

**前置要求：** Node.js 18+、梯子

=== "Windows"

    PowerShell 不走 Clash 的系统代理，安装前需先在同一个 PowerShell 窗口临时设置代理（端口以 Clash 实际显示为准，通常是 7897）：

    ```powershell
    $env:HTTP_PROXY="http://127.0.0.1:7897"
    $env:HTTPS_PROXY="http://127.0.0.1:7897"
    ```

    然后执行安装：

    ```powershell
    irm https://claude.ai/install.ps1 | iex
    ```

    或用 WinGet（WinGet 走系统代理，无需单独设置）：

    ```powershell
    winget install Anthropic.ClaudeCode
    ```

    !!! tip "代理设置仅对当前窗口有效"
        关闭 PowerShell 后代理自动失效，下次安装或更新时重新设置即可。

=== "macOS / Linux"

    ```bash
    curl -fsSL https://claude.ai/install.sh | bash
    ```

    或用 Homebrew：

    ```bash
    brew install --cask claude-code
    ```

安装完成后验证：

```powershell
claude --version
```

!!! warning "npm 安装方式已废弃"
    旧教程中常见的 `npm install -g @anthropic-ai/claude-code` 已被官方标记为 deprecated，建议使用上方的官方安装脚本。

## CC-Switch（API 提供商管理）

Claude Code 需要配置 API 提供商才能使用。手动编辑配置文件很麻烦，**CC-Switch** 提供了可视化界面，一键切换 API 提供商。

[![CC-Switch](https://img.shields.io/badge/GitHub-CC--Switch-181717?style=flat&logo=github&logoColor=white)](https://github.com/farion1231/cc-switch/releases)

**主要功能：**

- 内置 50+ 提供商预设（Anthropic 官方、各类中转服务）
- 一键切换，Claude Code 无需重启终端即可生效
- 统一管理 MCP 服务器、系统托盘快速切换
- 跨设备云同步配置

**Windows 安装：** 下载 `CC-Switch-vX.X.X-Windows.msi` 安装包或便携版 `.zip`

**基本使用：**

1. 打开 CC-Switch → 点击「Add Provider」
2. 从预设列表选择提供商（官方登录选 **Anthropic Official**）
3. 填入 API Key → 点击「Enable」启用
4. 切换后 Claude Code 立即生效，无需重启

**跳过登录确认：**

CC-Switch 设置 → **窗口行为** → 勾选「**跳过 Claude Code 初次安装确认**」

开启后，每次启动 Claude Code 会自动跳过登录步骤，直接进入。如果之后又遇到要求登录的情况，关闭该选项再重新打开即可。

!!! tip "其他模型提供商"
    CC-Switch 不只能切换 Anthropic 官方 API，还内置了 DeepSeek、GLM（智谱）、Kimi 等国内模型提供商预设，以及各类国内中转服务商，无需梯子，价格通常更低。

## 配置 API 提供商

Claude Code 本身不绑定特定提供商，通过 CC-Switch 选择你的 API 来源即可。

**没有 Anthropic 订阅？** 在 CC-Switch 中选择国内提供商（DeepSeek、GLM、Kimi 等），无需梯子，价格更低，跳过下方步骤直接进入「第一次使用」。

---

### 获取 Anthropic API Key（订阅用户）

已订阅 Claude Pro / Max 的用户才需要此步骤：

1. 访问 [![Anthropic Console](https://img.shields.io/badge/Anthropic-console.anthropic.com-CC785C?style=flat)](https://console.anthropic.com)（需要梯子）
2. 用与 claude.ai 相同的账号登录
3. 进入 **API Keys** → **Create Key**
4. 复制生成的 Key（以 `sk-ant-` 开头），**只显示一次，立即保存**
5. 在 CC-Switch 中选择 **Anthropic Official**，填入 Key 并启用

!!! info "关于订阅"
    Claude 只有**订阅制**，没有按量充值。免费账号仅能使用网页版，无法创建 API Key。详见 [Claude 订阅方案](订阅方案.md)。

## 第一次使用

如果终端提示找不到 `claude` 命令，需要将安装目录加入系统 PATH，详见 [添加环境变量 PATH](../../03-工具与问题/Windows常见问题/添加环境变量PATH.md)。

在你的项目目录下运行 `claude`，进入交互模式：

```bash
cd /path/to/your/project
claude
```

**常用指令：**

```
# 直接描述任务
> 帮我写一个Python脚本，读取文件并统计每行的字符数

# 让它读取并分析代码
> 看一下 main.py，解释这段代码的逻辑

# 修复 bug
> 这个函数运行报错 TypeError: ..., 帮我找原因并修复

# 做 CTF
> 这是一道 SQL 注入题，URL 是 http://..., 帮我分析注入点
```

## 常用技巧

**给足上下文：** Claude 看到的上下文越多，回答越准确。告诉它你的目标、当前错误信息、相关代码片段。

**迭代式工作：** 复杂任务分步完成，每步确认结果再继续，而不是一次性交出所有任务。

**CTF 场景：** 把题目描述、附件内容、已知信息都提供给 Claude，让它帮你分析思路和写 exploit。
