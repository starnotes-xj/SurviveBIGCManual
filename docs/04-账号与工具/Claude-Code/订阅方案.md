# Claude 订阅方案

Claude 采用**订阅制**，没有按量充值选项。免费账号仅能使用 claude.ai 网页版，**无法使用 API 和 Claude Code**。

[![claude.ai 订阅页面](https://img.shields.io/badge/claude.ai-订阅方案-CC785C?style=flat-square)](https://claude.ai/upgrade)

---

## 方案对比

| 方案 | 价格 | 用量 | 适合人群 | Claude Code | API Key |
|---|---|---|---|---|---|
| **Free** | 免费 | 基础额度 | 轻度体验 | ❌ 不支持 | ❌ 不支持 |
| **Pro** | $20/月 | Free 的 **5×** | 日常使用 | ✅ 支持 | ✅ 可创建 |
| **Max 5x** | $100/月 | Pro 的 **5×** | 重度开发 | ✅ 支持 | ✅ 可创建 |
| **Max 20x** | $200/月 | Pro 的 **20×** | 全天候开发 | ✅ 支持 | ✅ 可创建 |

!!! info "用量说明"
    用量限额按滚动窗口计算（每 5 小时一个周期），并设有每周上限。Max 并非无限使用，只是额度更高。绝大多数用户用 Pro 就够了，频繁触发限制时再考虑升级 Max 5x。

!!! tip "限时福利：每周限额 +50%（至 2026 年 7 月 13 日）"
    Anthropic 于 2026 年 5 月 13 日宣布，所有付费计划（Pro、Max、Team、Enterprise）的 Claude Code **每周用量限额临时提升 50%**，无需任何操作自动生效，到期后恢复正常。可在 Claude Code 中输入 `/usage` 查看当前用量。

## 如何订阅

国内用户直接在 claude.ai 网页端用银行卡订阅往往会失败，推荐通过 **Google Play 内购**完成订阅，成功率高。

### 前置准备

1. **工商银行 Visa 卡**：用于绑定 Google Pay。详见 [工商银行 Visa 卡办理](../../05-校园生活/银行卡办理/工商银行Visa卡.md)
2. **Google 账号**：用于登录 Google Pay 和 Claude App。详见 [Google 账号注册](../Google账号/注册与常见问题.md)
3. **梯子**：全程需要保持开启

### 订阅步骤

**第一步：将 Visa 卡绑定到 Google Pay**

在电脑浏览器上操作：

1. 访问 [![Google Pay](https://img.shields.io/badge/Google-Pay-4285F4?style=flat-square&logo=googlepay&logoColor=white)](https://pay.google.com)，登录 Google 账号
2. 左侧菜单选择「**付款方式**」→「**添加付款方式**」
3. 选择「信用卡/借记卡」，填入工商银行 Visa 卡卡号、有效期、CVV
4. 按提示完成验证（工商银行会发短信验证码）

**第二步：安装 Google Play 商店并下载 Claude App**

国内 Android 手机通常没有预装 Google Play，需要先安装。Google 官方没有提供独立下载页，推荐从 APKMirror 下载（每个文件均经人工审核）：

[![Google Play Store on APKMirror](https://img.shields.io/badge/APKMirror-Google_Play_Store-1A73E8?style=flat-square&logo=android&logoColor=white)](https://www.apkmirror.com/apk/google-inc/google-play-store/)

安装前需在手机设置中开启「允许安装未知来源应用」，安装完成后登录 Google 账号，再搜索并安装 Claude（开发商为 **Anthropic PBC**，注意认准，避免下载到同名第三方仿冒 App）：

[![Claude on Google Play](https://img.shields.io/badge/Google_Play-Claude_by_Anthropic-414141?style=flat-square&logo=googleplay&logoColor=white)](https://play.google.com/store/apps/details?id=com.anthropic.claude&hl=en_US)

**!鸿蒙系统**系统说明请查看：[Gbox 安装教程](./GBox解决部分安卓手机无法使用Google Play问题.md) 

[![GBox](https://img.shields.io/badge/GBox-官网-4CAF50?style=flat-square&logo=android&logoColor=white)](https://gboxlab.com/)

**第三步：订阅**

1. 打开 Claude App，用 **Google 账号**登录
2. 点击左上角菜单 → 「**Upgrade plan**」
3. 选择 Pro（$20/月）或 Max 套餐
4. 确认使用 Google Pay 付款，完成购买

!!! tip "为什么推荐 Google Play 订阅"
    Claude 官网直接用国内银行卡付款经常被拒，而 Google Play 内购走 Google Pay 通道，绑定 Visa 卡后成功率高很多。

## 订阅后创建 API Key

订阅 Pro 或 Max 后，可以在 Anthropic Console 创建 API Key，用于 Claude Code：

1. 访问 [![Anthropic Console](https://img.shields.io/badge/Anthropic-console.anthropic.com-CC785C?style=flat-square)](https://console.anthropic.com)
2. 登录与 claude.ai 相同的账号
3. 进入 **API Keys** → **Create Key**
4. 复制生成的 Key（以 `sk-ant-` 开头），**只显示一次，立即保存**

## 不想订阅官方怎么办

可以在 **CC-Switch** 中切换到其他模型提供商（如 DeepSeek、GLM、Kimi 等），或国内中转服务商，无需梯子，价格通常更低。
