# 学校教育邮箱申请

学校教育邮箱（`学号@stu.bigc.edu.cn`）是申请多项学生优惠的关键，包括 Notion Education、GitHub Education、JetBrains 学生认证等。

!!! tip "没有教育邮箱？先用学信网报告解锁 JetBrains 和 GitHub"
    JetBrains 和 GitHub Education 都接受**学信网在线验证报告**作为替代凭证，**不用等教育邮箱申请下来就能用**。登录 [![学信网](https://img.shields.io/badge/学信网-chsi.com.cn-22c55e?style=flat)](https://www.chsi.com.cn/) 申请在线验证报告即可。详见 [JetBrains 许可证申请](JetBrains/许可证申请.md)。

!!! warning "北印教育邮箱申请较繁琐，但长期值得"
    教育邮箱需老师签字 → 学院书记审核 → 学校网络中心审核，流程慢。但 Notion Education Plus 等服务**只认教育邮箱**，无替代方案，申请成功后一劳永逸。

---

## 为什么教育邮箱这么重要

| 服务 | 教育邮箱能获得什么 |
|---|---|
| **GitHub Education** | GitHub Copilot 教育版免费（**自2026年4月20日**起Copilot Pro、Copilot Pro+ 和学生计划的新用户注册将暂时停止）、私有仓库无限制、Student Developer Pack |
| **Notion Education** | Education Plus 计划免费（否则 $12/月） |
| **JetBrains** | 全套 IDE（IDEA、CLion、PyCharm 等）免费 |
| **Figma / Autodesk / Microsoft Azure** | 官方宣称提供教育优惠，未亲测，建议自行前往官网确认 |

## 北京印刷学院教育邮箱申请流程

### 通过学校信息门户

1. 登录北京印刷学院统一身份认证系统 [![BIGC 信息门户](https://img.shields.io/badge/BIGC-信息门户-003DA5?style=flat&logo=googlechrome&logoColor=white)](https://ehall.bigc.edu.cn/)
2. 在可用应用中找到"**学生电子邮箱申请**"入口
3. 下载申请承诺书，需要找一个老师签字，证明材料用 JetBrains 申请教育许可证或者 GitHub 教育优惠申请的截图即可
4. 老师签完字之后上传，记得让老师联系学院书记审核通过，书记审核通过之后等待学校网络中心审核。审核通过后邮箱格式为 `学号@stu.bigc.edu.cn`，登录：[![邮箱登录](https://img.shields.io/badge/邮箱登录-stu.bigc.edu.cn-EA4335?style=flat&logo=gmail&logoColor=white)](https://mail.stu.bigc.edu.cn/)

## 申请到邮箱后——拿到学生优惠

### GitHub Education

[![GitHub Education](https://img.shields.io/badge/GitHub-Education_申请-2da44e?style=flat&logo=github&logoColor=white)](https://education.github.com/students)

1. 点击 **Get student benefits**
2. 添加学校教育邮箱，或上传学生证图片（两种验证方式）
3. 等待审核（1-3 个工作日）

→ 详见 [教育优惠申请](GitHub注册与使用/教育优惠申请.md)

### Notion Education

[![Notion Education](https://img.shields.io/badge/Notion-Education_申请-000000?style=flat&logo=notion&logoColor=white)](https://www.notion.so/product/notion-for-education)

1. 用教育邮箱注册或绑定已有账号
2. 即时生效，升级为 Education Plus

→ 详见 [Notion 教育优惠申请](Notion/教育优惠申请.md)

### JetBrains

[![JetBrains Education](https://img.shields.io/badge/JetBrains-Education_申请-000000?style=flat&logo=jetbrains&logoColor=white)](https://www.jetbrains.com/community/education/)

1. 点击 **Apply now** → 选择 **University email address**
2. 填写教育邮箱，验证后即可免费使用全套 IDE

→ 详见 [JetBrains 许可证申请](JetBrains/许可证申请.md)

## 没有教育邮箱怎么办？

部分服务支持其他验证方式：

| 服务 | 替代验证 |
|---|---|
| GitHub Education | **学信网在线验证报告截图**（推荐，学生证通过率低） |
| JetBrains | **学信网在线验证报告**（约 7 天审核）详见 [许可证申请](JetBrains/许可证申请.md) |
| Notion | 仅支持教育邮箱，无替代 |
