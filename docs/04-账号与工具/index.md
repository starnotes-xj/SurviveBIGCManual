# 账号与工具

这些是信息安全学习的基础设施，越早配好越省事。

## 推荐配置顺序

```
梯子 → GitHub → JetBrains → Google账号 → Notion / 教育邮箱 / Claude Code
```

梯子是其他一切的前提，学信网验证报告和教育邮箱(本校教育邮箱申请麻烦)是换取免费工具的钥匙。
