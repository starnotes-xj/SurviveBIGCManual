## 支付方式

ChatGPT Plus 在订阅页面支持直接输入 Visa/Mastercard 卡号完成支付，无需 PayPal：

1. 打开 [chat.openai.com](https://chat.openai.com)，点击左下角头像 → **Upgrade to Plus**
2. 在支付页面选择 **Credit / Debit Card** 或 **Link**
3. 填写以下信息：

| 字段 | 说明 |
|---|---|
| Card Number | 卡号（16 位） |
| Expiry Date | 有效期，格式 MM/YY |
| CVV/CVC | 背面签名栏旁的 3 位数字 |
| Billing Address | 填写国内真实地址 |

工商银行星座学生版 Visa 信用卡可直接使用，申请方式见 [工商银行 Visa 卡](../../05-校园生活/银行卡办理/工商银行Visa卡.md)。

!!! tip "付款前确认卡内有足够的美元余额"
    订阅前需先在工行 App 购汇，确保 Visa 卡内有至少 $20 的美元余额。

## 常见问题

**Q：支付失败怎么办？**

1. 确认卡内美元余额充足
2. 确认境外支付功能已开通
3. 部分 IP 地址会被 OpenAI 风控拒绝，尝试更换梯子节点（建议使用美国节点）
4. 账单地址填写真实国内地址即可，不需要填境外地址
