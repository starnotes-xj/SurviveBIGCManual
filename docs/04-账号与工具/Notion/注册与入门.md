# Notion 注册与入门

Notion 是一款集笔记、数据库、任务管理、Wiki 于一体的生产力工具，非常适合用来管理课程笔记、项目资料和学习计划。

---

## 注册账号

1. 访问 [![Notion](https://img.shields.io/badge/Notion-notion.so-000000?style=flat-square&logo=notion&logoColor=white)](https://www.notion.so)（需要梯子，或直接访问有时也能打开）
2. 点击 **Get Notion free**
3. 推荐用 **Google 账号**登录，省去单独记密码
4. 选择用途（学生/个人），进入工作区

---

## 界面基本概念

```
工作区（Workspace）
└── 页面（Page）
    ├── 子页面
    ├── 数据库（Database）
    │   ├── 表格视图（Table）
    │   ├── 看板视图（Board）
    │   ├── 日历视图（Calendar）
    │   └── 画廊视图（Gallery）
    └── 普通内容块（Block）
        ├── 文本、标题、列表
        ├── 代码块
        ├── 图片、文件
        └── 内嵌其他页面
```

## 适合学生的用法

=== "课程笔记"

    每门课建一个页面，用 `/toggle` 折叠列表整理知识点，用数据库管理作业截止日期。

    ```
    📚 计算机网络笔记
    ├── 第一章 概述
    ├── 第二章 物理层
    └── 期末复习
    ```

=== "CTF 解题记录"

    建立一个 CTF 题目数据库，记录：题目名、类型、难度、解题思路、参考链接。

    | 题目 | 类型 | 状态 | 解题思路 |
    |---|---|---|---|
    | Web签到 | Web | ✅ | robots.txt |
    | 古典密码 | Crypto | 🔄 | 凯撒 + 维吉尼亚 |

=== "项目管理"

    建一个任务看板，追踪项目进度。比 Excel 好看，比 Jira 简单。

=== "学习资源收集"

    把各类教程链接、PDF、截图集中存放，比浏览器书签更方便管理。

## 常用快捷键

| 操作 | 快捷键 |
|---|---|
| 新建块 | `Enter` |
| 插入命令菜单 | `/` |
| 标题 1/2/3 | `/h1` `/h2` `/h3` |
| 代码块 | `/code` |
| 待办清单 | `/todo` |
| 折叠列表 | `/toggle` |
| 搜索 | ++control+p++ / ++cmd+p++ |
| 全局搜索 | ++control+k++ / ++cmd+k++ |
