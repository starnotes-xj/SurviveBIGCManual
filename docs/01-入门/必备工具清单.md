## Windows 自制工具

PowerShell 原生缺少一些 Linux 常用命令，以下工具填补这些空缺，均由本手册作者开发。

| 工具 | 用途 | 下载 |
|---|---|---|
| `grep.exe` | 文本搜索，支持正则、递归、颜色高亮，用法与 Linux grep 一致 | [![下载](https://img.shields.io/badge/下载-grep.exe-181717?style=plastic&logo=github&logoColor=white)](https://github.com/starnotes-xj/SurviveBIGCManual/releases/download/tools-v1.0/grep.exe) |

!!! tip "安装方法"
    下载后将 `.exe` 放到任意目录，然后把该目录加入系统 `PATH`，之后就能在 PowerShell 和终端中直接使用。
    
    推荐放在 `C:\tools\` 或 `D:\快捷方式\` 之类固定位置。
