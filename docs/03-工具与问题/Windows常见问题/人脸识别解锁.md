# 普通摄像头人脸识别解锁

Windows Hello 的人脸解锁需要红外摄像头，大多数笔记本和台式机不支持。FaceWinUnlock-Tauri 通过注入自定义 Credential Provider DLL，让普通 USB/内置摄像头也能实现类似体验。

## 适用场景

- 电脑有摄像头，但不支持 Windows Hello 人脸识别
- 追求开机免密/快速解锁的个人开发机、家用 PC
- **不适合**：存放敏感数据的办公机或服务器

!!! danger "安全专业学生必读"
    本工具基于 **2D 人脸识别**，持一张照片即可绕过解锁——与 Windows Hello 的红外活体检测有本质区别。作为信息安全专业学生，你应该清楚这不是真正的身份认证，只是便捷体验的替代品。**在涉及实验数据、账号密码的机器上请慎用。**

## 安装注意事项

!!! warning "操作前务必看完"
    - 软件会修改**注册表**和 **Winlogon 进程**。配置错误时可能导致登录界面白屏/无法进入桌面（概率极小，但需提前留有备份）
    - 初始化时软件会锁屏并自动解锁，**不要手动干预**，等待 5 秒即可
    - 密码处填写**账户登录密码，不是 PIN 码**。很多用户习惯用 PIN，在这里填 PIN 会提示密码错误

### 系统要求

- Windows 10 / 11，64 位
- 可用的摄像头（内置或 USB 均可）

### 安装流程

1. 下载并解压安装包，运行主程序
2. 按照程序内向导完成「系统初始化」（建议对初始化界面拍照留档）
3. 录入面容（支持多账户、多张面容）
4. 重启验证

## 卸载

程序内置「维护与卸载」功能，一键清除 DLL 注入和注册表修改，无需手动清理。

!!! tip "如果登录界面出现异常"
    进入 Windows 恢复环境（开机时按住 Shift + 重启），用命令提示符手动还原注册表，或使用初始化时的留档照片对照恢复。
