# CTF 学习资源

## 入门平台

### picoCTF

[![picoCTF](https://img.shields.io/badge/CMU-picoCTF-4E9BCD?style=plastic)](https://picoctf.org/)

卡内基梅隆大学出品，**绝对新手首选**。题目从最简单开始，每步都有提示，游戏化体验无痛入门。平台常年开放，可以随时练习历年题目。

## 国内练习平台

国内平台题目量大、有中文社区、网络访问稳定，是日常刷题的主战场。

| 平台 | 特点 |
|---|---|
| [![BUUCTF](https://img.shields.io/badge/BUUCTF-buuoj.cn-0078D4?style=plastic)](https://buuoj.cn) | 题库最大，收录大量真实赛题，各方向均有，**日常刷题首选** |
| [![ctf.show](https://img.shields.io/badge/ctf.show-ctf.show-FF6B6B?style=plastic)](https://ctf.show) | Web 方向题目质量极高，有配套视频讲解 |
| [![NSSCTF](https://img.shields.io/badge/NSSCTF-nssctf.cn-00BCD4?style=plastic)](https://www.nssctf.cn) | 界面现代，在线 Docker 环境，新题更新快 |
| [![攻防世界](https://img.shields.io/badge/攻防世界-xctf.org.cn-E53935?style=plastic)](https://adworld.xctf.org.cn) | XCTF 官方平台，赛题档案丰富，有新手任务路线 |

## 实战靶场

两个最值得推荐的实战平台，侧重真实攻防而非解题，适合有一定基础后系统提升：

| 平台 | 说明 |
|---|---|
| [![TryHackMe](https://img.shields.io/badge/TryHackMe-tryhackme.com-212C42?style=plastic)](https://tryhackme.com) | **引导式学习路径**，按主题分章节带你从零入门，每个房间配有详细讲解，浏览器内直接开靶机，**无需本地配置环境**，友好度极高 |
| [![HackTheBox](https://img.shields.io/badge/HackTheBox-hackthebox.com-9FEF00?style=plastic&logo=hackthebox&logoColor=black)](https://www.hackthebox.com) | **真实机器渗透**，Starting Point 新手村免费且引导完善，后续 Active Machines 难度较高，全球排名活跃，是渗透测试方向的标杆平台 |

## 赛事信息

[![CTFtime](https://img.shields.io/badge/CTFtime-ctftime.org-FF6D00?style=plastic)](https://ctftime.org/)

CTF 赛事"信息中心"。查找全球比赛时间、查看历年赛题存档、阅读官方 Writeup。参加比赛前先在这里确认赛制和难度。

## 推荐路线

1. **零基础**：从 picoCTF 开始，建立对各方向的感性认识
2. **系统入门**：TryHackMe 按学习路径刷房间，无需配环境直接上手
3. **选择方向**：Web → ctf.show；Pwn → pwn.college；Crypto → CryptoHack；综合 → BUUCTF
4. **日常刷题**：BUUCTF 或 NSSCTF，遇到不会的题找 Writeup，看懂后独立复现
5. **实战提升**：HackTheBox Starting Point → Active Machines，体验真实渗透流程
6. **参加比赛**：在 CTFtime 找合适的新手赛报名，赛后读 Writeup 复盘

!!! tip "关于 Writeup"
    不会的题看 Writeup 是正常的学习方式，但一定要自己动手复现一遍。光看不练等于没学。
