## 如何使用 Writeup

Writeup 是比赛结束后选手对自己解题过程的详细记录，包含：

- **思路分析**：如何发现漏洞或解题方向
- **利用过程**：完整的命令、脚本或操作步骤
- **关键知识点**：涉及的技术原理

!!! tip "学习建议"
    先自己尝试做题，实在卡住再查 Writeup。直接看答案会跳过最有价值的思考过程。看完 Writeup 后，建议自己动手复现一遍，而不是只读文字。
