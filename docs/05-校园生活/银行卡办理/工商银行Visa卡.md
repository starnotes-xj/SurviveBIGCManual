# 工商银行 Visa 卡办理

Visa 卡用于**境外支付**，信息安全学习中常用到：

- 订阅 ChatGPT Plus / Claude Pro
- 购买境外域名（Namecheap、GoDaddy）
- 注册 AWS / Google Cloud / Azure
- 购买安全工具授权（Burp Suite Pro 等）
- GitHub Copilot 等付费服务

---

## 工行 Visa 借记卡 vs 信用卡

| | 借记卡（推荐） | 信用卡 |
|---|---|---|
| 申请条件 | 有工行储蓄账号即可 | 需要收入证明，学生较难申请 |
| 审核时间 | 1-2周 | 更长 |
| 额度 | 账户余额上限 | 银行授予信用额度 |
| 汇率 | Visa 实时汇率 | 同左 |
| 年费 | 通常免年费 | 有年费（部分可减免） |

**学生首选借记卡（牡丹灵通卡国际版）。**

## 办理流程

### 第一步：开立工行账户

如果已经有工行储蓄卡，可跳过此步。

前往工行网点，带身份证开立储蓄账户。

### 第二步：申请 Visa 借记卡

1. 携带 **身份证** 和 **现有工行储蓄卡** 前往工行网点
2. 告知柜员："我要申请一张带 Visa 功能的借记卡，用于境外支付"
3. 填写申请表，留下手机号
4. 柜员会告知审核周期（通常 1-2 周）
5. 卡片邮寄到你指定地址，或到网点领取

### 第三步：激活

收到卡后，通过工行 App 或 ATM 激活：

- 工行 App：首页 → 我的账户 → 找到新卡 → 激活
- ATM：插卡 → 按提示激活

---

## 开通境外支付

激活后，还需要开通境外支付功能：

1. 打开工行 App → 我的 → 我的银行卡 → 选择 Visa 卡
2. 找到"境外支付"或"跨境交易"选项 → 开通
3. 设置每日境外交易限额（根据需要设置，默认可能较低）

---

## 充值与使用

向 Visa 卡账户存入人民币，支付时系统自动按当日汇率换算为外币。

**订阅境外服务时填写信息：**

- Card Number：卡号（16位）
- Expiry Date：有效期（印在卡面，格式 MM/YY）
- CVV/CVC：背面签名栏旁的3位数字
- Billing Address：填写国内地址即可（部分服务需要，填真实地址）

## 常见问题

**Q：支付被拒了怎么办？**

1. 检查境外支付是否已开通
2. 检查余额是否充足（含汇率换算后的金额）
3. 检查每日限额是否超出
4. 部分平台不支持中国发行的卡，此时考虑使用虚拟信用卡（如 Depay、OneKey Card）

**Q：有没有年费？**

工行牡丹灵通卡（借记卡）通常免年费，办理时确认一下。

**Q：Visa 卡和普通借记卡是同一张吗？**

是的，工行会给你一张卡，但这张卡同时具备银联（国内消费）和 Visa（境外消费）功能，看卡面上有没有两个 logo。
