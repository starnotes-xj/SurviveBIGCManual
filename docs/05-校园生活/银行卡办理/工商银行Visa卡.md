# 工商银行星座学生版信用卡

这是一张专门面向在校学生的信用卡，带有 Visa 功能，可用于境外支付，是信息安全学习中的常用工具：

- 订阅 ChatGPT Plus / Claude Pro
- 购买境外域名（Namecheap、GoDaddy 等）
- 注册 AWS / Google Cloud 等云平台
- GitHub Copilot、Burp Suite Pro 等付费服务

学生版信用卡不需要收入证明，凭学生身份即可申请，是目前学生获取 Visa 卡最方便的方式。

---

## 办理流程

可以在工行 App 或网点直接申请，线上填写更方便：

1. 打开**工行 App**，搜索「星座学生版信用卡」，或前往工行网点告知柜员
2. 选择自己喜欢的**星座图案**（12 星座可自选）
3. **选卡地点必须选「北京」**，否则无法匹配学校地址
4. 填写工作/学校信息，单位名称填：**北京印刷学院信息工程学院**
5. 填写本人真实信息（身份证号、手机号等）
6. 提交后基本秒过审，卡片通过**邮政物流**寄出（1-2 周）
7. 主校区的快递在**东门（正大门）保安室正对面的收发室**领取
8. 收到卡后**前往工商银行网点线下激活**，首次办理需到柜台找工作人员办理

!!! tip "选卡地点记得选北京"
    这一步容易漏掉，系统默认可能是户籍所在地，一定要手动改为「北京」。

---

## 激活后：充值外汇

学生版信用卡为 **0 额度**，不预存人民币，付款前必须先在卡上存入对应币种的外汇。

**第一步：向信用卡充入人民币**

信用卡还款（充值）支持以下两种方式，均需提前绑定银行卡：

=== "支付宝"

    1. 打开支付宝 → 进入「**余额宝**」
    2. 余额宝中必须有钱，没有的话先充入 **0.01 元**
    3. 点击「**转出**」→ 选择「**还款**」
    4. 选择工商银行星座学生版信用卡
    5. 输入金额，选择付款方式，完成支付

=== "微信"

    打开微信 → 服务 → 搜索「**信用卡还款**」→ 选择工商银行信用卡 → 输入金额还款

!!! info "两张卡共享余额"
    星座学生版信用卡可能附带银联和 Visa 两张卡，但它们属于同一账户，余额共享。只需还款一次，两张卡均可使用。

**第二步：换汇（工行 App）**

1. 打开工行 App → 搜索「**结售汇**」
2. 选择「购汇」，选择币种（美元 USD 最常用）
3. 填写金额，交割方式选「**无需取现**」（直接存入 Visa 卡账户）
4. 确认汇率后提交，到账后即可用于境外支付

!!! warning "付款前确认余额"
    每次付款前确认卡内有足够的对应币种余额，余额不足会直接被拒。

## 绑定支付平台

推荐绑定以下两个平台，基本覆盖所有境外支付场景：

**PayPal**

国际主流支付方式，ChatGPT、各类境外服务均支持。绑定流程需要完成验证：

1. 在 PayPal 添加工行 Visa 卡后，PayPal 会向卡扣除 **$1.95** 作为验证金
2. 等待 **3-5 天**，账单出来后在工行 App 账单明细中找到该笔记录
3. 账单备注中有一串**验证码**，将其输入到 PayPal 的验证框中
4. 验证通过后解除限制，扣除的 $1.95 **会在之后返还**

**Google Pay**

绑定后可在 Google Play 订阅服务（如 Claude App），详见 [Claude 订阅方案](../../04-账号与工具/Claude-Code/订阅方案.md)

## 开通境外支付

线下激活后**默认已开通境外支付**，无需额外操作。

如需调整每日境外交易限额，可在工行 App → 我的 → 我的银行卡 → 选择该信用卡 → 境外支付设置中修改。

---

## 使用方法

订阅境外服务时填写以下信息：

| 字段 | 说明 |
|---|---|
| Card Number | 卡号（16 位） |
| Expiry Date | 有效期，格式 MM/YY，印在卡面 |
| CVV/CVC | 背面签名栏旁的 3 位数字 |
| Billing Address | 填写国内真实地址即可 |

## 常见问题

**Q：支付被拒了怎么办？**

1. 确认境外支付功能已开通
2. 确认信用额度充足
3. 确认每日限额未超出
4. 部分境外平台不支持中国发行的卡，此时可考虑虚拟卡（如 Depay、OneKey Card）

**Q：有没有年费？**

学生版信用卡通常有年费优惠政策，办理时向柜员确认免年费条件（通常达到一定刷卡次数即可免）。

**Q：和普通信用卡有什么区别？**

星座学生版专为在校学生设计，申请门槛低、额度适中，毕业后可申请升级为普通版。
