# 北京银行卡办理

入学后需要办理一张**北京银行**的借记卡，奖学金、助学金、学校补贴等款项均通过北京银行发放，这张卡是必须要有的。

!!! warning "注意是「北京银行」"
    北京银行（Bank of Beijing）是一家独立的银行，不是"在北京办的银行卡"。工商银行、中国银行等都不符合要求，必须专门去办北京银行的卡。

---

## 所需材料

- 身份证原件（必须携带）
- 学生证（建议携带）
- 手机号（办卡时绑定）

!!! tip "刚开学没有学生证怎么办"
    大一新生入学后几周才会发放学生证。如果没带学生证，可以给工作人员看**微信班级群**作为在校学生的证明，通常可以正常办理。

---

## 在哪里办

大一新生在**康庄校区**，推荐优先去**黄村分行**办理，距离近且人少，办理速度快。

开学期间北京银行工作人员也会来学校设点办理，但排队人多，等待时间长，**不推荐**在学校摆摊时办理，自行前往网点更省时。

---

## 办理流程

1. 前往北京银行黄村分行（可在北京银行 App 或微信小程序查询具体地址）
2. 取号 → 等待 → 告知柜员「办理借记卡，学生」
3. 填写开卡申请表，提供手机号
4. 设置取款密码（6 位数字）
5. 领卡并当场激活
6. 下载「北京银行」App，绑定登录

---

## 办卡后

**第一步：绑定微信/支付宝**

- 微信：我 → 服务 → 钱包 → 银行卡 → 添加银行卡
- 支付宝：我的 → 银行卡 → 添加银行卡

**第二步：在学校系统登记卡号（必须操作）**

1. 微信搜索并关注「**北印收费**」服务号
2. 进入「**智慧校园服务**」
3. 点击「**微信登录**」
4. 进入「**个人中心**」→「**个人资料**」
5. 填写手机号和北京银行卡卡号，保存

登记后饭补会自动每月打入该卡，无需其他操作。

!!! info "饭补发放"
    每月 **25 号左右**发放到北京银行卡：
    
    - 非贫困生：**66 元 / 月**
    - 贫困生：**536 元 / 月**

---

## 常见问题

**Q：还需要再办其他银行的卡吗？**

日常消费用北京银行卡就足够了。如果有境外支付需求（订阅 ChatGPT、Claude 等），可以参考 [工商银行 Visa 卡办理](工商银行Visa卡.md)。

**Q：学生卡和普通卡有区别吗？**

开卡时主动告知是学生，通常可办免年费的学生卡。

**Q：可以网上开卡吗？**

实体卡需要到柜台办理，但可以在**北京银行 App 提前预约**，选择黄村分行，预约后到店可以减少等待时间，省事很多。
