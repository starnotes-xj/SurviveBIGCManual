# 北京银行卡办理

在北京上学，需要一张本地银行卡用于日常消费、微信/支付宝绑定、校园内转账等。

---

## 选哪家银行

对学生来说，以下几家最常用：

| 银行 | 推荐指数 | 说明 |
|---|---|---|
| 中国银行 | ★★★★★ | 校园网点多，手续简单，长城借记卡可绑定支付宝/微信 |
| 工商银行 | ★★★★ | ATM 最多，功能全，也可以办 Visa 卡（见另一篇文档） |
| 招商银行 | ★★★★ | App 好用，年轻人多用，但部分城市开卡需预约 |
| 建设银行 | ★★★ | 稳妥，功能齐全，无特别亮点 |

**推荐首选：中国银行或工商银行。** 两家在学校附近通常都有网点，开卡流程最简便。

---

## 所需材料

- 身份证原件（必须携带）
- 学生证（部分银行要求，建议带上）
- 手机号（办卡时绑定，用于短信通知）

---

## 办理流程

1. 携带材料前往银行网点（推荐提前查好营业时间，避免午休时段）
2. 取号 → 等待 → 告知柜员"办理借记卡"
3. 填写开卡申请表，提供手机号
4. 设置取款密码（6位数字）
5. 领卡，当场激活（柜员会引导）
6. 下载对应银行 App，绑定手机号登录

---

## 常见问题

**Q：需要存最低余额吗？**

大部分借记卡无最低余额要求，但部分卡种若长期余额为零可能产生小额管理费。开卡时问清楚。

**Q：学生卡和普通卡有什么区别？**

学生卡通常免年费、免小额账户管理费，主动要求办学生卡或告知对方是学生。

**Q：可以网上开卡吗？**

部分银行支持 App 开电子账户，但功能有限。要办实体卡还是需要到柜台。

---

## 开卡后

- 微信：我 → 服务 → 钱包 → 银行卡 → 添加银行卡
- 支付宝：我的 → 银行卡 → 添加银行卡
- 设置每日转账/取款限额（建议根据需要调整，默认限额可能偏低）
