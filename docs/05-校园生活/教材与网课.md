# 教材与网课

## 教材

除了**英语**等需要扫描二维码激活才能使用的教材外，其他课程的教材基本可以不购买——一学期下来实际翻开的次数屈指可数。

**替代方案（按推荐顺序）：**

1. **电子版**：许多教材有 PDF 版本，优先找电子版使用。
2. **学长学姐二手教材**：价格更低，部分还附有现成的笔记和划重点，性价比极高。
3. **借用同学的**：同班同学互相借阅也是常见做法。

!!! tip "如何找到二手教材"
    学长学姐毕业季会在院内群、二手交易平台（如闲鱼）出售教材，留意群消息或主动询问即可。新生群、年级群里也经常有人转让。

## 网课

### 大一下：人工智能导论

该课程为网课形式，视频较多。推荐使用 **Global Speed** 浏览器扩展以 **16 倍速**播放，可大幅缩短刷课时间。

**安装方式：**

[![Chrome Web Store](https://img.shields.io/badge/Chrome_Web_Store-Global_Speed-4285F4?style=flat&logo=googlechrome&logoColor=white)](https://chromewebstore.google.com/detail/global-speed-video-speed/jpbjcnkcffbooppibceonlgknpkniiff)

支持 Chrome、Edge、Firefox。

### 通识选修课（学习通网课）

教务系统里选择的课程默认均为**线下课**，但大一下起可以在**学习通**上另行选择网课作为通识选修课。

**选课建议：**

- 优先选择**学分为 1 分**的网课，内容少、刷课快。
- 通识选修课共需修满 **5 学分**，每学期可选 **2 门**网课。
- 按照「每学期 2 门 × 1 学分」的节奏，从大一下开始选，到**大二下即可刷完全部 5 学分**。

使用 **OCS** 可以自动刷课，安装方式见[必备工具清单 → Windows 效率工具](../01-入门/必备工具清单.md#windows-效率工具)。OCS 配置题库后还能**自动答题**，基本无需人工干预。

!!! warning "注意事项"
    - 刷课适用于进度类课程，期末若有线上考试仍需自行作答。
    - 使用前确认所选课程平台受 OCS 支持（学习通、智慧树等主流平台均已支持）。
